# Description
Workout Buddy, is a workout tracker web app focusing on simplicity and a clean design. ​Its workout log lets you view daily workouts easily. Add an exercise to the workout log, set weight of the exercise and the number of reps to perform. The exercise is added to the workout log with a timestamp. The user may delete the workout exercise once they have completed it.

# Features
- simple and clean design
- user can track their workouts easily
- timestamps provided to show when each workout was created
- the user may delete the workout as per their needs
- the workouts are stored in a MongoDB database and can be accessed even after closing the application

### How to Setup:
- Step 1: Download the zip
- Step 2: open two terminals
- Step 3: In first terminal, cd into frontend. In second terminal cd into backend
- Step 4: run 'npm install' on both terminals
- Step 5: run 'npm start' on both terminals
